# Login-Page

A basic login page built with **HTML**, **CSS**, and **Express.js** for learning purpose.

## Features
- Responsive login form
- Styled with CSS
- Server-side handling with Express.js

## Installation & Usage
1. Clone the repository:
```bash
git clone https://github.com/yourusername/simple-login-page.git
```
2. Navigate to the project folder:

```bash
cd Login-page
```
3. Install dependencies:

```bash
npm install
```
4. Start the server:

```bash
node server.js
or
nodemon server.js (if you have that)
```
5. Open your browser at http://localhost:3000 and test the login page.
